`Vue.js` 官方 [[ESLint]] 代码检查插件。

检查范围：
- `.vue` 后缀文件中的 `<templete>` 和 `<script>` 标签。
- `.js` 文件中的 `vue` 代码。

检查内容：
- 语法错误
- 违反 [Vue 官方风格指导](https://vuejs.org/style-guide/)的写法
- 错误使用 [官方内置指令](https://vuejs.org/api/built-in-directives.html)

命令行检查：
```bash
npx eslint src
```
