# Mac
## 图标与按键映射表
| 图标      | 按键 |
| --------- | ---- |
| command   | ⌘    |
| option    | ⌥    |
| control   | ⌃    |
| shift     | ⇧    |
| caps lock | ⇪    |

**查看分辨率** : `cmd` + `space`，搜索`系统报告`
**emoji** : `ctrl` + `cmd` + `space`
新建文件夹:`shift`+`cmd`+`n`
重命名: 选中后按下`return`
